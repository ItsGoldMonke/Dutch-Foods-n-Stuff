# Dutch Foods n Stuff

This mod adds a bunch of typical Dutch things and things associated with the Dutch.

**It currently adds:**
- Cheese
- Cheese blocks
- Cheese slices & slicer

**Going to be added:**
- Clogs
- Tullip clusters
- Stroopwafel
- Bitterballen
- **_More_**
